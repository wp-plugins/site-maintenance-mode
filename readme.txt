== Description ==

Простой плагин для добавления возможности переключения сайта на "Режим обслуживания".

Особенности:

*   Настраивается как текст сообщения (HTML), так и CSS-стили
*   Для администраторов добавляется виджет на страницу "Консоль"
*   Настраиваются категории пользователей, для которых следует ограничить доступ к сайту


Supported languages:

*   Russian (RU)
*   English (EN)
*   Deutsch (DE)

== Installation ==

1. Загрузить плагин в директорию `/wp-content/plugins/`
2. Активировать на вкладке 'Плагины'
3. Настроить и включать при необходимости

2. Страница административной панели
3. Виджет для страницы страницы "Консоль"
